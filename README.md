# ⛪ Plataforma Web - Iglesia ACyM Crucero

Plataforma Web Institucional (SPA) desarrollada para la Iglesia Alianza Cristiana y Misionera, congregación de Crucero, Río Bueno, Chile.

## 🚀 Características
- **Navegación Interactiva SPA:** No recarga la página, haciendo la navegación instantánea.
- **Responsive:** Adaptada 100% a computadoras, tablets y celulares.
- **Calendario Dinámico:** Base de datos en JSON con versículos bíblicos individuales.
- **Contador Inteligente:** Calcula en tiempo real cuántos días faltan para el próximo cumpleaños.
- **Buscador & Filtros:** Permite buscar hermanos por nombre o filtrar por mes específico.
- **Tailwind CSS:** Diseño moderno utilizando Glassmorphism (Efecto cristal) y colores institucionales.

## 🛠️ Tecnologías
- HTML5 / CSS3 / JavaScript (Vainilla)
- Tailwind CSS (vía CDN)
- FontAwesome (Íconos)
- Entorno Node.js (opcional, para desarrollo local)

## 📁 Instrucciones para Imágenes y Logo
Para que las imágenes carguen correctamente en GitHub:
1. Sube tu imagen del logo a la carpeta `/assets/img/`.
2. Asegúrate de que se llame **exactamente** `Logo_Crucero_Acym.png` (respetando mayúsculas y extensión).
3. Si cambias la imagen de "Nuestra Historia", llámala `image_11231e.jpg`.
4. Añade tu icono para la web en `favicon.ico`.

## 💻 Desarrollo Local (Node.js)
1. Clona el repositorio.
2. Abre la terminal en la carpeta raíz.
3. (Opcional) Ejecuta `npm start` (requiere Node.js > 24) para iniciar un servidor local y probar la página antes de publicar los cambios en GitHub.
